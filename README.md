gocover
=======
```
Usage: gocover [-l] [-c]
   or: gocover [options] (FILE-NUMBER|FILENAME)...

Utility reads the coverage profile and shows annotated source code. You
can create coverage profile, list files in it and then get highlighted
source code.

Options:
      --color-cover string     set color for covered code (default "green")
      --color-header string    set color for header (default "yellow")
      --color-uncover string   set color for not covered code (default "red")
  -c, --cover                  run tests with the coverage
  -f, --file string            the coverage profile (default "coverage.out")
  -h, --help                   show this text and exit
  -l, --list                   list files in the coverage profile
  -V, --version                output version information and exit

Report bugs to author.

```
